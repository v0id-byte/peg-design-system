Stacked property list — bold term followed by a muted em-dash definition.

```jsx
<AttrList items={[
  { term: 'Stable by design',           definition: 'pegged to $1' },
  { term: 'Fully backed',               definition: 'a redeemable claim on reserves', accent: true },
  { term: 'Earns the issuer, not you',  definition: 'pays no interest to holders' },
  { term: 'Spends like cash, 24/7',     definition: 'but no legal tender or deposit insurance' },
]} />
```

**Rule:** 3–5 items max per slide. One accent item (jade) per list — mark the defining advantage or risk.
